## 说明
使用vue-cli3.0构建项目,实现的一个pizza点餐系统,管理页面使用导航守卫需要登录
1. 主要使用vuex管理各个组件件的数据及状态,
2. 数据存储使用的是野狗云,
3. 数据请求使用的axios,

## 使用
```
git https://github.com/zhangwj0520/pizza-app
cd node-app
npm install
npm run serve
npm run build
